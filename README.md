# NETFLIX-clone
making a clone of netflix using react js

### Tech stacks used :
MERN stands for : M - MongoDB, E - Express, R - React, N - Node.js

To create this clone following Tech stack is used by contributors.

*  **React** 
*  **Express** 
*  **Node.js** 
*  **JSX** 
*  **React Router Dom** 
*  **material-ui/core** 
*  **axios** 
*  **material-ui** 
*  **react-dom** 
*  **react-scripts**
*  **dotenv** 
*  **cors** 
*  **jsonwebtoken**
